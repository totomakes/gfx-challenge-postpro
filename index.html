The workflow

Four stages, start to finish

Each stage shows its steps and the file it produces. Click to expand.

  • Producer backs up RAW and builds proxies with the Blackmagic Proxy Generator; proxies + sound recordings go to the editors.
  • Cut on a sequence that matches the core spec exactly.
  • Drop grantee JPGs as temporary placeholders where final images will go.
  • Publish review versions for each feedback round.
Produces — Review version 4K · H.264 · .mov · 20,000 kbps → Frame.io Recipe A →
  • Send the image list of every still used in the cut to the Fuji team.
  • Receive full-resolution finals + the matching metadata graphics (PNG overlays).
  • Confirm every final is sRGB. Convert any other color space to sRGB before placing — keeps colors identical and prevents mismatches.
  • Place each metadata PNG directly on top of its image, for the full time it’s on screen.
  • Export an XML so post pulls and archives only the RAW actually used.
Color management: JPGs were placeholders only. If finals arrive as Adobe RGB, Display P3 or ProPhoto, convert to sRGB before they hit the timeline.

∥ parallel Audio handoff

  • Export an AAF with embedded clips — every channel / line separated for sound design.
  • Export a 1080p H.264 reference (.mp4 · 20,000 kbps) so they can sync to picture.

You said “AAC” — the format that carries separated tracks + embedded clips is AAF. AAC is a final mixdown codec and can’t hold separate lines, so the handoff is an AAF.

∥ parallel Color

  • Colorist grades from the conform and returns a single ProRes 422 HQ file the editor lays straight in.

then Online / approval

  • Lay back color + final mix + graphics and build the full graphics stack.
  • Export the online version and upload to Frame.io for sign-off.
Produces — Online (approval) 4K · H.264 · .mov · 20,000 kbps → Frame.io Recipes A · B · C · E →
Master 1 — Clean

No GFX / VFX over the images across the whole cut, and no subtitles. Keep the intro & outro.

Master 2 — Full

Everything on — bug, letterbox, metadata overlays, subtitles, intro & outro.

Produces — Masters ×2 ProRes 422 HQ · .mov → Frame.io or Dropbox Recipe E →

Proxy
Lightweight copy of the RAW for fast editing; conformed back for finishing.
Picture lock
Edit is final — no more timing or shot changes. Triggers finals, audio & color.
Bug
Persistent Fujifilm logo (hidden over full-res images and the intro).
Metadata graphic
PNG overlay riding on top of each final image, from the Fuji team.
AAF
Audio interchange carrying separated tracks with embedded clips for sound design.
ProRes 422 HQ
Mastering codec for the color return and both final masters.
sRGB
Required color space for all final images. Convert anything else first.

Reassembling color + audio + graphics at full quality for approval.
